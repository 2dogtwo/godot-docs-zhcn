Pin joint for 3D PhysicsBodies.

Descripción
----------------------

Pin joint for 3D rigid bodies. It pins 2 bodies (rigid or static) together. See also :ref:`Generic6DOFJoint<class_Generic6DOFJoint>`.

enum **Param**:

- **PARAM_BIAS** = **0** --- La fuerza con la que los objetos clavados se mantienen en relación posicional entre sí. Cuanto más alto, más fuerte.

- **PARAM_DAMPING** = **1** --- La fuerza con la que los objetos clavados se mantienen en relación de velocidad entre sí. Cuanto más alto, más fuerte.

- **PARAM_IMPULSE_CLAMP** = **2** --- If above 0, this value is the maximum value for an impulse that this Joint produces.
