Clase base para todas las mallas primitivas. Maneja la aplicación de un :ref:`Material<class_Material>` a una malla primitiva. Los ejemplos incluyen :ref:`CapsuleMesh<class_CapsuleMesh>`, :ref:`CubeMesh<class_CubeMesh>`, :ref:`CylinderMesh<class_CylinderMesh>`, :ref:`PlaneMesh<class_PlaneMesh>`, :ref:`PrismMesh<class_PrismMesh>`, :ref:`QuadMesh<class_QuadMesh>`, y :ref:`SphereMesh<class_SphereMesh>`.

| *Setter*  | set_flip_faces(value) |
+-----------+-----------------------+
| *Getter*  | get_flip_faces()      |

If set, the order of the vertices in each triangle are reversed resulting in the backside of the mesh being drawn.

This gives the same result as using :ref:`SpatialMaterial.CULL_BACK<class_SpatialMaterial_constant_CULL_BACK>` in :ref:`SpatialMaterial.params_cull_mode<class_SpatialMaterial_property_params_cull_mode>`.

.. _class_PrimitiveMesh_method_get_mesh_arrays:

- :ref:`Array<class_Array>` **get_mesh_arrays** **(** **)** |const|

Devuelve los conjuntos de malla utilizados para constituir la superficie de :ref:`Mesh<class_Mesh>`. El resultado puede ser pasado al :ref:`ArrayMesh.add_surface_from_arrays<class_ArrayMesh_method_add_surface_from_arrays>` para crear una nueva superficie. Por ejemplo:

::

    var c = CylinderMesh.new()
    var arr_mesh = ArrayMesh.new()
    arr_mesh.add_surface_from_arrays(Mesh.PRIMITIVE_TRIANGLES, c.get_mesh_arrays())
