TreeItem
========

**Inherits:** :ref:`Object<class_Object>`

Control para un solo elemento dentro de un :ref:`Tree<class_Tree>`.

Descripción
----------------------

Control para un solo elemento dentro de un :ref:`Tree<class_Tree>`. Puede tener hijos ``TreeItem``\ s y ser estilizado así como contener botones.

Puedes quitar un ``TreeItem`` usando el :ref:`Object.free<class_Object_method_free>`.
