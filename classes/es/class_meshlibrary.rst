MeshLibrary
===========

**Inherits:** :ref:`Resource<class_Resource>` **<** :ref:`Reference<class_Reference>` **<** :ref:`Object<class_Object>`

Biblioteca de mallas.

Descripción
----------------------

Una biblioteca de mallas. Contiene una lista de recursos de :ref:`Mesh<class_Mesh>`, cada uno con un nombre y una identificación. Cada elemento también puede incluir formas de colisión y de navegación. Este recurso se utiliza en :ref:`GridMap<class_GridMap>`.

--------------------

- `3D Platformer Demo <https://godotengine.org/asset-library/asset/125>`__
