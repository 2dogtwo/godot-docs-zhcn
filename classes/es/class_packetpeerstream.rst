PacketPeerStream
================

**Inherits:** :ref:`PacketPeer<class_PacketPeer>` **<** :ref:`Reference<class_Reference>` **<** :ref:`Object<class_Object>`

Envoltura para usar un PacketPeer sobre un StreamPeer.

Descripción
----------------------

PacketStreamPeer proporciona un envoltorio para trabajar usando paquetes sobre un flujo. Esto permite usar código basado en paquetes con StreamPeers. PacketPeerStream implementa un protocolo personalizado sobre el StreamPeer, por lo que el usuario no debe leer o escribir en el StreamPeer envuelto directamente.
