TranslationServer
=================

**Inherits:** :ref:`Object<class_Object>`

Servidor que gestiona todas las traducciones.

Descripción
----------------------

Servidor que gestiona todas las traducciones. Las traducciones pueden ser configuradas y eliminadas de él.
