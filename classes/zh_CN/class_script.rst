Description
-----------

作为资源存储的类。脚本可以扩展所有以它为实例的对象的功能。

这是所有脚本的基类，不应直接使用。尝试使用此类创建新脚本将导致错误。

脚本子类的 ``new`` 方法会创建新的实例。如果一个现有的对象的类与脚本的基类之一相匹配，\ :ref:`Object.set_script<class_Object_method_set_script>` 会扩展该对象。
