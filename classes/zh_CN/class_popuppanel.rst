PopupPanel
==========

带有面板背景的弹出窗口。

.. rst-class:: classref-introduction-group

Description
-----------

带有可配置面板背景的弹出窗口。添加到这个节点的子控件都会被拉伸到面板的大小（类似于 :ref:`PanelContainer<class_PanelContainer>` 的原理）。如果你制作的是窗口，请参阅 :ref:`Window<class_Window>`\ 。
