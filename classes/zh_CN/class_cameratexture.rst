CameraTexture
=============

.. rst-class:: classref-introduction-group

Description
-----------

该纹理可以访问 :ref:`CameraFeed<class_CameraFeed>` 提供的相机纹理。

\ **注意：**\ 许多相机提供的都是 YCbCr 图像，需要在着色器中进行转换。
