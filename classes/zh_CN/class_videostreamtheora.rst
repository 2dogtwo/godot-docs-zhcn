Description
-----------

:ref:`VideoStream<class_VideoStream>` 资源处理带有 ``.ogv`` 扩展名的 `Ogg Theora <https://www.theora.org/>`__ 视频格式。Theora 编解码器在 CPU 上进行解码。

\ **注意：**\ 虽然 Ogg Theora 视频也可以具有一个 ``.ogg`` 扩展名，但必须将该扩展名重命名为 ``.ogv``\ ，才能在 Godot 中使用这些视频。
