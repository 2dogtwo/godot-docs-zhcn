将一种色调应用于一张画布的节点。

.. rst-class:: classref-introduction-group

Description
-----------

**CanvasModulate** 将一种色调应用于一张画布上的所有节点。一张画布只有一个可用于为画布着色，但 :ref:`CanvasLayer<class_CanvasLayer>` 可用于独立渲染事物。
