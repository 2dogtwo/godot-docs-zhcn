Description
-----------

2D 矩形，旨在用于物理学。通常用于为 :ref:`CollisionShape2D<class_CollisionShape2D>` 提供形状。

\ **性能：**\ **RectangleShape2D** 可以快速检测碰撞。比 :ref:`CapsuleShape2D<class_CapsuleShape2D>` 快，但比 :ref:`CapsuleShape2D<class_CapsuleShape2D>` 慢。
