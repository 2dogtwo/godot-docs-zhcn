RootMotionView
==============

**Inherits:** :ref:`VisualInstance3D<class_VisualInstance3D>` **<** :ref:`Node3D<class_Node3D>` **<** :ref:`Node<class_Node>` **<** :ref:`Object<class_Object>`

在 :ref:`AnimationTree<class_AnimationTree>` 中设置根运动的仅编辑器可用的辅助工具。

.. rst-class:: classref-introduction-group

Description
-----------

*根运动*\ （Root Motion）是一种动画技术，通过使用网格的骨架为角色提供冲量。在处理 3D 动画时，动画师通常使用根骨骼来为其余的骨骼提供动作，从而使得角色的动画能够准确地匹配地面，并实现在电影中与物体的精确交互。另请参阅 :ref:`AnimationTree<class_AnimationTree>`\ 。

\ **注意：**\ **RootMotionView** 仅在编辑器中可见。在运行的项目中将自动隐藏。
