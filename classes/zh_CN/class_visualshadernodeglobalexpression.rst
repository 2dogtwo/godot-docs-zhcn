Description
-----------

自定义Godot着色器语言表达式，位于生成的着色器之上。你可以在内部放置各种函数定义，以便以后在\ :ref:`VisualShaderNodeExpression<class_VisualShaderNodeExpression>`\ 中调用，这些函数被注入到主着色器函数中。你还可以声明varyings、uniforms 和全局常量。
