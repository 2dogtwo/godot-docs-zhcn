代表 GLTF 纹理采样器

.. rst-class:: classref-introduction-group

Description
-----------

表示由基本 GLTF 规范定义的纹理采样器。GLTF 中的纹理采样器指定在对象上渲染纹理时，如何从纹理的基础图像中采样数据。
