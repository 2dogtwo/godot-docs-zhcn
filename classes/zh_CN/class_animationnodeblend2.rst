Description
-----------

添加到 :ref:`AnimationNodeBlendTree<class_AnimationNodeBlendTree>` 的资源。根据取值将两个动画进行线性混合。

一般而言，混合值应该在 ``[0.0, 1.0]`` 的范围内。超出该范围的值可以混合放大或翻转后的动画，然而，这种场合 :ref:`AnimationNodeAdd2<class_AnimationNodeAdd2>` 的效果更好。
