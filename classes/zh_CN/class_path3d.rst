Path3D
======

Description
-----------

可以让 :ref:`PathFollow3D<class_PathFollow3D>` 子节点沿着 :ref:`Curve3D<class_Curve3D>` 移动。有关用法的更多信息，请参见 :ref:`PathFollow3D<class_PathFollow3D>`\ 。

注意，该路径被认为是相对于移动的节点（\ :ref:`PathFollow3D<class_PathFollow3D>` 的子节点）。因此，该曲线通常应以零向量 ``(0, 0, 0)`` 开始。
