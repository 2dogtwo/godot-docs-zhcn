Description
-----------

用于确认动作的对话框。这个窗口类似于 :ref:`AcceptDialog<class_AcceptDialog>`\ ，但按下“取消”按钮和按下“确定”按钮的效果是不同的。这两个按钮的顺序取决于主机操作系统。

要获得取消操作，你可以使用：


.. tabs::

 .. code-tab:: gdscript

    get_cancel_button().pressed.connect(self.canceled)

 .. code-tab:: csharp

    GetCancelButton().Pressed += Canceled;
